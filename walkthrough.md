# MVP Demo Walkthrough

## How to Run the Demo

1.  **Navigate to the codebase folder**:
    ```bash
    cd "d:\EPF COURSE\Engineering Project\codebase"
    ```

2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the Streamlit App**:
    ```bash
    streamlit run app.py
    ```
